# Blog Platform 🚀

A full-stack MERN blog application with authentication, image upload, likes, edit/delete, and protected routes.

## 🔥 Features

- User Registration & Login
- JWT Authentication
- Create Blog with Image Upload
- Edit/Delete Blog
- Like System
- Protected Routes
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
